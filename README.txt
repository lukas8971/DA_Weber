# README for writing a thesis in LaTeX
#
################################################################################

Use this template to write a bachelor or master's thesis or a dissertation.

This template is based—with slight adjustments—on the vutinfth document class.
For more information see README-vutinfth.txt.
Changes from the original:
* an additional INSO specific title page is inserted
* the main document file name is thesis.tex rather than example.tex
* for Unix-based systems, in addition to build-all.sh there is also a Makefile.
    make clean compile view
* additional organizational information, toc, additional package(s)


For general help about using the templates and for help with LaTeX, please consult 
the top-level README.txt file.
